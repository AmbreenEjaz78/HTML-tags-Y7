# HTML Tags Interactive Worksheet
Topic Structure & Presentation in HTML Tier 1 — Support Total Marks 20

🎯 Learning Intention

Students will be able to identify and use common HTML tags to structure and format a web page.

1 I can recognise what HTML tags are and why we use them.
2 I can match HTML tags to their correct purpose.
3 I can write simple HTML using the correct tags.
